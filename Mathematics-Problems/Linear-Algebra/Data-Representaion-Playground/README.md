# Data Representation Playground

## Overview
This project demonstrates how data is represented in Machine Learning using
scalars, vectors, matrices, and tensors.

## Concepts Covered
- Scalars (hyperparameters)
- Vectors (feature representation)
- Matrices (datasets)
- Tensors (images and batches)
- Shape manipulation

## Why This Matters
Understanding data shapes is critical for:
- Model design
- Debugging ML pipelines
- Neural networks and deep learning

## Key Learnings
- ML models operate purely on numerical data
- Shape mismatches cause most beginner errors
- Tensors generalize vectors and matrices
