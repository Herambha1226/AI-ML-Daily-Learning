-> Handling Large CSV (For example : 1M+ Rows)


-> Use Chunks : A chunk is a DataFrame representing a specific block of rows from your CSV file.

->  for chunk in pd.read_csv("large.csv", chunksize=10000):
        print(chunk.shape)

-> Use specific Columns : 

->  pd.read_csv("large.csv", usecols=["name", "age"])

-> Specify data types: for using reduces memory usage.

-> pd.read_csv("large.csv", dtype={"age": "int32"})


-> Format Conversion (Very Important) <-

-> CSV → JSON:

df.to_json("data.json", orient="records")

-> CSV → Excel

df.to_excel("data.xlsx", index=False)

-> CSV → Parquet (Parquet is faster and compressed)

df.to_parquet("data.parquet")
